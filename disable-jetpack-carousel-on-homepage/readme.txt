Disables Jetpack Carousel on homepage.

== Description ==

[Carousel](http://jetpack.me/support/carousel/) is a fantastic gallery lightbox feature included with the [Jetpack](http://wordpress.org/plugins/jetpack/) plugin. This plugin disables the carousel behaviour on the homepage.

== Installation ==

Please see [Installing Plugins](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins) in the WordPress Codex.

No configuration is necessary. Just install then activate and Carousel will be disabled on the home page automatically.
